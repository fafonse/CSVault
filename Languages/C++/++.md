The ++ operator adds one to a variable.
# Integers
## Postfix
When used after the variable, it adds 1 to the number and returns the number **before it added one**.
```cpp
int x = 5;
cout << x++ << endl; // prints 5
cout << x << endl; // prints 6
```

## Prefix
When used before the variable it increments the variable before the expressions used.
```cpp
int x = 5;
cout << ++x << endl; // prints 6
cout << x << endl; // prints 6
```
