Typically placed after the data segment of a datagram, it is also referred to as *redundancy*.


## Parity checking
Single bit parity:
- Detect single bit errors
- Can only detect an odd number of bit errors
	- If there's two bit errors, it won't detect it.

Two-dimensional bit parity:
- detect and *correct* single bit errors

## Cyclic Redundancy Check
A more powerful error-detection coding.
A super awesome checksum sort of function.

> Typically used in Ethernet and WiFi
