The reporting and monitoring hub for AWS. 
It's automatically turned on, and you ***will*** get charged for the storage those logs take.

## Print Statements
If you want to see any print statements made by your API:
1. Cloudwatch
2. Logs
3. Log Management
4. \<program name\>
5. Select log

You'll see a bunch of stuff in there, but your print statements will be on their own line.
## Removing Unnecessary Logs
This can change constantly, but here are the steps as of 2026-02-24:
1. Cloudwatch
2. Logs
3. Log management
4. Select \<program name\>
	- Or just click it's link
5. Actions drop down menu
6. Edit Retention Settings
