
## Types of requirements
1. Functional requirements
	- Requirements for the software to actually work as intended
		- EX: API's, actual function
- Non-functional Requirements
	- Requirement for the program to work for the user
		- EX: RAM usage, uptime, storage space
- User Stories
- Personas
