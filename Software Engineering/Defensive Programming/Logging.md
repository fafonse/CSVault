## Developing For Customers

If you're doing logging and you need to share it with customers, make sure that you **collect the data before you report it**.
You can always generate reports on past data, but you cannot generate a report if you don't have data already collected.

1. Collect data
	- **Collect as much as possible**, especially if you don't have a report software ready
2. Filter it
3. Connections 
4. Reporting
