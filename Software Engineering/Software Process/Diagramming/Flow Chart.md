A diagram that shows how data is processed and flows. Specifically shows event sequences and their branches.

Flow charts are also *super useful* for spotting failure 

Useful for:
- User flows
- Data processing
- AI Agent flows

> Does not show all the systems/processes. Just a snapshot of a dynamic dataflow. For representing the system as a whole, use [[Component Diagram|component diagrams]].
## Style

- Branches should be *diamonds/boxes*
- End-states/dead ends are *circles*
- Arrows are ONE DIRECTION
	- Caller $\rightarrow$ Callee
	- Add context to the lines for more detail
