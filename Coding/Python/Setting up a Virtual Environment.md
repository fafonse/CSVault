1. Navigate to your project folder
2. `python3 -m venv "venv folder name"`
	- I like to use `.venv` as my folder name
3. To activate your virtual environment it depends on your OS
	- Linux
		- `source .venv/bin/activate`
	- Windows
		- `.venv\Scripts\activate`
	- Windows Powershell
		- `.venv\Scripts\Activate.ps1`
4. When you're done, you can leave the `.venv` environment with
	- `deactivate`
